# Shopping Cart Demo Application

A demo application showcasing the Shopping Cart Library functionality.

## Installation

```bash
npm install
```

## Running

```bash
npm run dev
```

The demo will run on `http://localhost:3000`

## Features Demonstrated

- Browse products
- Add products to cart with quantity
- View cart via CartWidget
- Update product quantities
- Remove items
- Select items for checkout
- View cart totals

## Project Structure

- `src/main.jsx` - Entry point
- `src/App.jsx` - Main application component
- `src/App.css` - Application styles

## Building

```bash
npm run build
```
